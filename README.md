## Getting Started

1. Clone Repository
```
git clone https://github.com/themewagon/symposium-nextjs.git
```
2. Install Dependencies
```
npm i
```
3. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Speedy Loan Finance Services is a modern, scalable, and high-performance financial services web application built using React and Vite, designed to digitize and streamline the loan consultation and customer onboarding process. The platform provides a comprehensive presentation of product-wise loan services, eligibility requirements, and detailed document structures while maintaining a clean, responsive, and professional user interface suitable for financial institutions, DSAs, and loan consultants.

The application is seamlessly integrated with WhatsApp and Email communication channels, enabling instant customer engagement, lead follow-ups, and real-time support directly from the website. It also supports Equifax credit report integration, allowing secure credit score retrieval and eligibility verification to accelerate loan decision-making and reduce manual processing. An interactive EMI Calculator is embedded within the platform to help users instantly calculate monthly installments, interest rates, and loan tenure, ensuring transparency and improving user confidence before applying for loans.

Built with performance and maintainability in mind, the project leverages Vite’s fast build system, reusable React components, and a scalable folder architecture. The website is fully responsive, optimized for SEO, and designed to deliver a smooth user experience across all devices. This project serves as a complete digital solution for showcasing loan products, managing customer interactions, and supporting credit evaluation workflows in a modern financial services environment.
